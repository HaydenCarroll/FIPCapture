# FIPCapture
Allows a captured region of your screen to be displayed onto a Logitech/Saitek flight instrument panel.

To use please make sure you have the official Logitech fip drivers installed before launching the program.
https://support.logi.com/hc/en-au/articles/360024848713--Downloads-Flight-Instrument-Panel
